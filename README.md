Projeto: API de Gerenciamento de Contatos

API desenvolvida com o objetivo de praticar os fundamentos do desenvolvimento backend.

Tecnologias: .NET, Entity Framework Core.

Funcionalidades: Implementação completa de métodos CRUD (Create, Read, Update, Delete).

Foco: Estudo de mapeamento objeto-relacional (ORM), manipulação de dados e arquitetura de APIs RESTful.
